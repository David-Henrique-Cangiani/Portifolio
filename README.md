# Portfólio — David Henrique Cangiani

Portfólio pessoal responsivo apresentando minha atuação como **AI Product Builder**: automações, integrações de IA e aplicações web. Construído em HTML, CSS e JavaScript.

🔗 **No ar:** [david-henrique-cangiani.github.io/Portifolio](https://david-henrique-cangiani.github.io/Portifolio/)

## Seções

- **Início** — posicionamento e stack principal
- **Sobre** — quem sou e como trabalho
- **Projetos** — trabalhos reais publicados (IA Mode Music, DHC WebSolutions e mais), com preview via iframe
- **Automação & IA Aplicada** — ferramentas de IA e automação
- **Processo** — da descoberta ao deploy
- **Contato** — e-mail, LinkedIn e GitHub

## Destaques técnicos

- Layout responsivo e moderno
- Grid de cards com animações em CSS
- Previews de projetos ao vivo via iframe
- Tipografia com Google Fonts e ícones Font Awesome

## Tecnologias

- HTML5, CSS3, JavaScript
- ScrollReveal (animações)
- Deploy via GitHub Pages

## Como rodar localmente

```bash
git clone https://github.com/David-Henrique-Cangiani/Portifolio.git
cd Portifolio

# Abra index.html no navegador, ou:
npx serve .
```
